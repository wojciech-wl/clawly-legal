4. Numery telefonów

Numer telefonu jest podawany opcjonalnie. Jest natychmiast hashowany (SHA-256) — surowy numer nie jest przechowywany na serwerze. Hash służy wyłącznie do jednorazowego dopasowania kontaktów. Użytkownik może wyłączyć wyszukiwalność po numerze w ustawieniach.

5. Gdzie przechowujemy dane

Dane przetwarzane są na terenie Unii Europejskiej (Google Cloud Platform, region europe-west1). Używamy: Google Cloud Firestore, Google Cloud Storage, Firebase Authentication, Firebase Cloud Messaging.

6. Udostępnianie danych

Nie udostępniamy danych podmiotom trzecim w celach marketingowych. Dane przetwarzane są przez Google Cloud Platform (hosting), Firebase (autoryzacja, push) i Apple (Sign-In).

7. Okres przechowywania

Dane przechowywane są do momentu usunięcia konta. Po usunięciu konta dane są oznaczane jako usunięte, a następnie trwale usuwane (profil, posty, relacje, dane z Firebase Auth).

8. Jak usunąć konto i dane

Aby usunąć konto i wszystkie powiązane dane:

  1. Otwórz aplikację Clawly
  2. Przejdź do Ustawienia (ikona w prawym górnym rogu)
  3. Przewiń na dół i kliknij Usuń konto
  4. Potwierdź usunięcie

Usunięcie obejmuje: profil właściciela, wszystkie profile kotów, posty, relacje follow, dane autoryzacji. Proces jest nieodwracalny.

Możesz też napisać na danielzielinski1234567@gmail.com z prośbą o usunięcie danych.
